Question2 Diff btw Inline , internal and external Css.
Inline CSS
* Applied to a specific HTML element. * Uses the style attribute in HTML element. * Has the higest priority and overrides internal and external style .
Internal CSS
* Applied to a Header Section. * Uses the style attribute in HTML element. * Has the 2nd higest priority and overrides external style . * Good for the single page.
External CSS
* stored in a specific file and linked to multiple Html doc. . * Uses the LINK attribute in external CSS file. * Good for large projects

Question5 What is z- index ? The z-index property in CSS controls the stacking order of elements along the z-axis in a 2D space. It determines which element appears in front or behind other elements when they overlap. It only works on elements with position: relative, absolute, fixed, or sticky (not static).
